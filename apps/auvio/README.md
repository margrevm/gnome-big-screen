# auvio

## Service

RTBF Auvio is RTBF's streaming platform for live TV, replay, and on-demand Belgian content.
Website: https://www.rtbf.be/auvio

## Usage

From this app directory:

```bash
./install.sh   # Install the app
./remove.sh    # Remove the app
```

## Limitations

- Relies on `chromium-browser` being available.
