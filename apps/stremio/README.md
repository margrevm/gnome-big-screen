# stremio

## Service

Stremio is a media center app for discovering and streaming movies, series, and channels.
Website: https://www.stremio.com/

## Usage

From this app directory:

```bash
./install.sh   # Install the app
./remove.sh    # Remove the app
```

## Limitations

- Stremio does not launch in full-screen ([#14](https://github.com/margrevm/gnome-big-screen/issues/14))
