# vrt-max

## Service
VRT MAX is VRT's streaming service for Flemish TV programs, series, and live channels.
Website: https://www.vrt.be/vrtmax/

## Usage

From this app directory:

```bash
./install.sh   # Install the app
./remove.sh    # Remove the app
```

## Limitations
- Relies on `chromium-browser` being available.
