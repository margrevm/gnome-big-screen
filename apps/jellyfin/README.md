# jellyfin

## Service

Jellyfin is an open-source media server for streaming your personal library.
Website: https://jellyfin.org/

This app installs and launches Jellyfin Media Player (`com.github.iwalton3.jellyfin-media-player`) via Flatpak.

## Usage

From this app directory:

```bash
./install.sh   # Install the app
./remove.sh    # Remove the app
```
