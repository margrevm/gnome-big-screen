# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project follows Semantic Versioning.

## [0.1.0] - 2026-03-03

### Added

- Initial public release of GNOME Big Screen setup scripts.
- Core installer (`install.bash`) and shared helpers (`common.bash`).
- Default launchers for Chromium and Shutdown.
- Optional app installers and uninstallers for:
  - ARTE
  - RTBF Auvio
  - VRT MAX
  - YouTube (VacuumTube)
  - Spotify
  - Stremio
  - Home Assistant
  - Jellyfin
  - Netflix
- App-level documentation and contribution guidelines.
